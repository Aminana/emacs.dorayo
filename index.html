Welcome to emcas.dorayo.

This is an Emacs auto config project including 'install' file, Emacs configure files(.emacs links to .emacs_simple or .emacs_default) and related plug-in components.

This project is now only support unix-like OS. You can use the command below to download this project and auto config your emacs23.

$ cd ~/
$ git clone https://github.com/dorayo/emacs.dorayo.git
$ cd emacs.dorayo
$ ./install
